# HR-Dashboard-
A comprehensive HR Analytics Dashboard built with Power BI, leveraging Power Query for data transformation and DAX for advanced calculations and insights. This dashboard provides clear visualizations of key HR metrics to support data-driven decision-making.
Project Overview
This repository contains:

The Power BI report file (.pbix)

A cleaned and transformed dataset ready for analysis

Documentation of the DAX measures and Power Query steps

Visual insights into employee demographics, retention, promotions, satisfaction, and more
